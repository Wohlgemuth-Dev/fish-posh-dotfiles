Einrichtung eines großartigen Linux-Terminals

Diese Anleitung führt Sie durch die Anpassung Ihres Ubuntu-Terminals, um es mit Fish Shell und Oh My Posh großartig aussehen zu lassen.

Fish Shell installieren

Öffnen Sie Ihr Terminal und führen Sie den folgenden Befehl aus:

sudo apt install fish -y

Standard-Shell zu Fish Shell ändern

Führen Sie den folgenden Befehl aus, um Fish Shell zu Ihrer Standard-Shell zu machen:

chsh -s /usr/bin/fish

Fish Shell ausführen

Um Fish Shell zu verwenden, geben Sie einfach Folgendes ein:

fish

Oh My Posh installieren

Laden Sie Oh My Posh herunter und installieren Sie es:

sudo wget https://github.com/JanDeDobbeleer/oh-my-posh/releases/latest/download/posh-linux-amd64 -O /usr/local/bin/oh-my-posh
sudo chmod +x /usr/local/bin/oh-my-posh

Schriftarten herunterladen und installieren

Erstellen Sie ein Verzeichnis für Schriftarten:

mkdir -p $HOME/.local/share/fonts

Laden Sie die FiraCode Nerd Font herunter und entpacken Sie sie:

wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/FiraCode.zip -O $HOME/Downloads/firacode.zip
unzip $HOME/Downloads/firacode.zip -d $HOME/.local/share/fonts

Aktualisieren Sie die Schriftartdatenbank:

fc-cache -f -v

Ändern Sie die Monospace-Textart Ihres Systems in "FiraCode Nerd Font Mono". Sie können dies mit GNOME Tweaks tun. Wenn Sie es nicht installiert haben:

sudo apt install gnome-tweaks -y

Oh My Posh Theme und Fish Config von GitHub herunterladen

Da Sie ein benutzerdefiniertes Theme und eine benutzerdefinierte Konfiguration verwenden, laden wir diese von Ihrem GitHub-Repository herunter.

Erstellen Sie zuerst die notwendigen Verzeichnisse:

mkdir -p ~/.poshthemes ~/.config/fish

Laden Sie dann die Dateien mit wget herunter. Bitte ersetzen Sie die URLs unten durch die tatsächlichen URLs für Ihre Dateien:

  • Oh My Posh Theme-Datei: https://github.com/Wohlgemuth-Dev/fish-posh-dotfiles/raw/main/customMontys.omp.json
  • Fish Config-Datei: https://github.com/Wohlgemuth-Dev/fish-posh-dotfiles/raw/main/config.fish

Mit wget:

wget https://github.com/Wohlgemuth-Dev/fish-posh-dotfiles/raw/main/customMontys.omp.json -O ~/.poshthemes/customMontys.omp.json
wget https://github.com/Wohlgemuth-Dev/fish-posh-dotfiles/raw/main/config.fish -O ~/.config/fish/config.fish

Oh My Posh Theme zur Fish Config hinzufügen

Die config.fish-Datei aus Ihrem Repository sollte bereits die Zeile zum Initialisieren von Oh My Posh enthalten. Lassen Sie uns dies jedoch überprüfen.

Öffnen Sie Ihre Fish-Konfigurationsdatei:

gedit $HOME/.config/fish/config.fish

Stellen Sie sicher, dass sie die folgende Zeile (oder eine ähnliche Zeile, die korrekt auf Ihre Theme-Datei verweist) enthält:

oh-my-posh init fish --config ~/.poshthemes/customMontys.omp.json | source

Wenn sie nicht vorhanden ist, fügen Sie sie hinzu. Wenn sie vorhanden ist, können Sie die Datei schließen.

Terminal-Farbschema ändern

bash -c "$(wget -qO- https://git.io/vQgMr)"

Geben Sie 69 ein, um (Everforest Dark Hard) auszuwählen.

Wichtig: GNOME Terminal lädt Fish Shell möglicherweise erst als Standard-Shell, nachdem Sie sich ab- und wieder angemeldet haben. Nachdem Sie die Schritte abgeschlossen haben, melden Sie sich von Ihrer aktuellen Sitzung ab und wieder an. Starten Sie dann GNOME Terminal.
