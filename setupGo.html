Go (Golang) und GoLand unter Ubuntu installieren

Diese Anleitung führt Sie durch die Installation der Programmiersprache Go (Golang) und der integrierten Entwicklungsumgebung (IDE) GoLand unter Ubuntu.

1. Go (Golang) installieren

Es gibt verschiedene Möglichkeiten, Go unter Ubuntu zu installieren. Die empfohlene Methode ist die Verwendung des offiziellen Tarballs.

  • Besuchen Sie die offizielle Go-Downloadseite: https://go.dev/dl/
  • Suchen Sie die neueste stabile Version für Linux (normalerweise eine .tar.gz-Datei).
  • Kopieren Sie die URL des Tarballs.
  • Öffnen Sie Ihr Terminal und laden Sie den Tarball herunter (ersetzen Sie die URL durch die kopierte): 
    wget https://go.dev/dl/go1.24.3.linux-amd64.tar.gz

    Hinweis: Die Versionsnummer kann sich ändern. Überprüfen Sie die Downloadseite für die aktuelle Version.

  • Extrahieren Sie den Tarball nach /usr/local: 
    sudo tar -C /usr/local -xzf go1.24.3.linux-amd64.tar.gz

    Auch hier gilt: Passen Sie den Dateinamen an die heruntergeladene Version an.

  • Fügen Sie das Go-Binärverzeichnis zu Ihrem PATH hinzu.
    • Für Bash/Zsh: Öffnen Sie Ihre ~/.profile oder ~/.bashrc-Datei mit einem Texteditor: 
      nano ~/.profile
      oder 
      nano ~/.bashrc
      Fügen Sie am Ende der Datei die folgende Zeile hinzu: 
      export PATH=$PATH:/usr/local/go/bin
    • Für Fish Shell: Öffnen Sie Ihre Fish-Konfigurationsdatei (normalerweise ~/.config/fish/config.fish oder ~/.fishrc): 
      nano ~/.config/fish/config.fish
      oder 
      nano ~/.fishrc
      Fügen Sie am Ende der Datei die folgende Zeile hinzu: 
      set PATH /usr/local/go/bin $PATH

      Wichtig: Fish Shell verwendet Leerzeichen zur Trennung von Pfaden in der PATH-Variable. Stellen Sie sicher, dass Sie keine Doppelpunkte verwenden.

  • Speichern Sie die Datei und schließen Sie den Editor.
  • Wenden Sie die Änderungen an Ihre aktuelle Shell an:
    • Für Bash/Zsh: 
      source ~/.profile
      oder 
      source ~/.bashrc
    • Für Fish Shell: 
      source ~/.config/fish/config.fish
      oder 
      source ~/.fishrc
  • Überprüfen Sie die Installation, indem Sie die Go-Version abfragen: 
    go version

    Sie sollten eine Ausgabe ähnlich wie go version go1.22.4 linux/amd64 sehen.

2. GoLand herunterladen und installieren

GoLand ist eine kommerzielle IDE von JetBrains, die speziell für die Go-Entwicklung entwickelt wurde. Sie können eine kostenlose Testversion herunterladen oder eine Lizenz erwerben.

  • Besuchen Sie die offizielle GoLand-Downloadseite: https://www.jetbrains.com/go/download/
  • Klicken Sie auf die Schaltfläche "Download" für die Linux-Version. Dies lädt eine .tar.gz-Datei herunter.
  • Öffnen Sie Ihr Terminal und navigieren Sie zu dem Verzeichnis, in dem die Datei heruntergeladen wurde (normalerweise der Ordner Downloads): 
    cd ~/Downloads
  • Extrahieren Sie den Tarball. Ersetzen Sie GoLand-*.tar.gz durch den tatsächlichen Dateinamen: 
    tar -xzf GoLand-*.tar.gz
  • Navigieren Sie in den extrahierten Ordner (der Name enthält die GoLand-Version): 
    cd GoLand-*/bin
  • Führen Sie das Installationsskript aus: 
    ./goland.sh
  • Das GoLand-Setup-Fenster wird geöffnet. Folgen Sie den Anweisungen auf dem Bildschirm, um GoLand zu installieren. Sie können wählen, ob Sie Verknüpfungen erstellen möchten.
  • Nach der Installation können Sie GoLand über das Anwendungsmenü oder über das Terminal starten (wenn Sie beim Setup eine entsprechende Option gewählt haben).

3. GoLand für die Go-Entwicklung konfigurieren

Nach dem Start von GoLand müssen Sie möglicherweise das Go SDK konfigurieren.

  • Wenn Sie GoLand zum ersten Mal starten, werden Sie möglicherweise aufgefordert, das Go SDK anzugeben.
  • Klicken Sie auf "Configure" oder "Setup SDK".
  • Wählen Sie "+ Add New SDK..." und navigieren Sie zu dem Verzeichnis, in dem Go installiert ist (normalerweise /usr/local/go).
  • Wählen Sie das Go-Verzeichnis aus und klicken Sie auf "OK".
  • GoLand erkennt nun Ihr Go SDK und ist für die Go-Entwicklung bereit.
